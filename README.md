hshb_notifier_win
=================

Delphi source of Open Space Notifier App for windows desktop (systray)

Verwendete Komponenten
======================

Soundfiles
----------

* Von https://www.freesound.org/
* User "fin"
* Dateien "171578__fins__3-down-1.wav" und "171580__fins__3-up-1.wav"

Icons und Logos
---------------

* Von Mitgliedern und Gästen des Hackerspace Bremen e.V. erstellt

Delphi 7-Komponenten
--------------------

Der Quellcode wurde in Delphi 7 geschrieben und nutzt die folgenden Komponenten:
* Synapse (http://www.ararat.cz/synapse/)
* bomeoneinstance (keine Homepage mehr)
* lkJSON (http://sourceforge.net/projects/lkjson/)
* JEDI Visual Component Library (http://jvcl.delphi-jedi.org/)
* TJPVersionInfo (http://www.delphidabbler.com/software?id=verinfo)
* OpenSSL-Bibliotheken (http://www.openssl.org/)

Installer
---------

* InnoSetup (http://www.jrsoftware.org/isinfo.php)
